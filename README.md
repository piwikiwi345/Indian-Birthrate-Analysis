### Scrubbing and Cleaning the Data

First, I got rid of the notes at the bottom of the data in its .txt file format. Then, I removed the newline characters from each string. At this point, I had placed every line of the .txt file into its own item in a list. I placed the clean lines (as single strings) in a new list titled "clean"

```python
clean = []

for x in all_lines:
    y = x.rstrip('\n')
    clean.append(y)
```

Then, I split each line (which was a single string) using the '\t' as the splitter. 

```python
non_tab = []

for row in clean:
    x = row.split('\t')
    non_tab.append(x)
```

The first four columns contained repetitive information. I chose to delete columns 1,2, and 4, but leave column 3 because it contained the year of each row in a clean format (i.e. just '1960').

```python
for row in non_tab:
    del row[0]
    del row[0]
    del row[1]
```

Some of the column titles included commas which caused google sheets to think a single column title was meant for multiple columns. SO I used the replace() function to clean this up. 

```python
col_titles = []

for x in non_tab[0]:
    y = x.replace(',', '')
    col_titles.append(y)

del non_tab[0]
non_tab.insert(0, col_titles)
```

Finally, I exported the list into a .csv file. I calculated the number of columns in each row so that I could create a new line at the appropiate place (because I had removed the '\n' which was initially in the .txt file).

```python
print(f'Number of Columns: {len(non_tab[0])}')

row_counter = 0

with open('data/clean_data.csv', 'w') as f:
    for row in non_tab:
        for item in row:
            f.write(item + ',')

            # create rows every 52 values printed
            row_counter = row_counter + 1
            if row_counter % 52 == 0:
                f.write('\n')
```

I left the majority of empty data values in the .csv file as Google Sheets had no problems calculating aggregate statistics for the columns I wanted even with the empty data values.

## Analysis

### Aggregate Statistics

The average sex ratio at birth (M to F) from 1960-2020 is meant to illustrate how India has always been slightly male dominated. While the past decades have seen India have an increase in the proportion of males in the general population, that proportion has been going down for the past couple of years.

The minimum, maximum, and average urban population growth from 1961-2021 was meant to illustrate how the urban population growth has been declining, and has declined past the rate in 1961. The max rate occured in 1974. The average occurred in 1991. The minimum occurred most recently in 2021 (without data on 2022 and 2023). In combination, these statistics showcase the trend of declining urban population growth rates - most likely the cause of increasing population density in cities and an increasing cost of living in major cities.

The average and minimum female population proportion are meant to show how the proportion of females in India's population has always been below 50%, but is recently increasing. The proportion moves between the ranges of 48.4% and 48.21% and increases and decreases over decades.

The average and maximum female population proportion are meant to show how the proportion of females in India's population has always been over 50%, but is recently increasing. The proportion moves between the ranges of 51.8% and 51.6% and increases and decreases over decades.

I think these patterns show a healthy trend for India. I calculated these statistics on the condition that only data from years after 1979 was used. 1979 was the year that China implemented their one child family policy. During that time, Chinese families often kept sons and gave daughters away for adoption. Many of these orphans found homes abroad or outside of China's general population, creating an imbalance of male and females in the population. India, a country with a similar population size, had no similar restrictions, and over the past decades, has maintained a healthy balance of males and females in their population.
